    Webhook Base: https://n8n.srv1013903.hstgr.cloud/webhook/c285fc03-6b3a.../

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                                     WORKFLOW 1: IMAGEN                                  │
    │  POST /imagen                                                                           │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Validate payload (jobId, prompt)                                                   │
    │  ├─ HTTP Request → Higgsfield /v1/text2image/soul                                      │
    │  │  ├─ params: { prompt, width_and_height, enhance_prompt, seed }                     │
    │  │  └─ headers: Authorization Key HIGGSFIELD_ID:HIGGSFIELD_SECRET                      │
    │  ├─ Poll status (max 20 loops, interval 2.5s)                                          │
    │  │  └─ GET /v1/job-sets/{request_id}                                                   │
    │  ├─ Save to Supabase tracker_results                                                   │
    │  │  ├─ job_id, action="imagen", result_url, status="completed", metadata              │
    │  │  └─ HTTP POST → SUPABASE_URL/rest/v1/tracker_results                               │
    │  └─ Response: { ok: true, jobId, status, url }                                         │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~45s | Cost: ~$0.02

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                                    WORKFLOW 2: VIDEO                                    │
    │  POST /video                                                                            │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Generate base image (text2image, similar to Workflow 1)                            │
    │  ├─ Poll image until ready                                                             │
    │  ├─ HTTP Request → Higgsfield /v1/image2video/dop                                      │
    │  │  ├─ params: { prompt, model, input_images[{image_url}], enhance_prompt, seed }    │
    │  │  └─ Select model: 'dop-turbo' (rápido) o 'dop-preview' (calidad)                   │
    │  ├─ Poll video (max 30 loops, interval 5s) ← TARDA MÁS TIEMPO                         │
    │  ├─ Save to Supabase                                                                   │
    │  └─ Response: { ok: true, jobId, status, url }                                         │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~5-10m | Cost: ~$0.50-2.00

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                                   WORKFLOW 3: VOICE                                     │
    │  POST /voice                                                                            │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Validate texto, idioma, voz                                                        │
    │  ├─ HTTP Request → ElevenLabs /v1/text-to-speech/{voice_id}                           │
    │  │  ├─ headers: xi-api-key, Accept: audio/mpeg                                         │
    │  │  └─ body: { text, model_id="eleven_multilingual_v2", voice_settings }              │
    │  ├─ Receive MP3 binary (base64 encode)                                                 │
    │  ├─ Save to Supabase (data URL)                                                        │
    │  └─ Response: { ok: true, jobId, status, url (base64 dataURL) }                        │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~5-10s | Cost: ~$0.01

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                               WORKFLOW 4: PRESENTACIÓN                                  │
    │  POST /presentacion                                                                     │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Parse prompt → slides structure                                                    │
    │  ├─ Loop (generar cada slide):                                                         │
    │  │  ├─ Generate slide content (Claude API si needed)                                   │
    │  │  ├─ Generate slide image (Higgsfield text2image)                                    │
    │  │  └─ Compile slide HTML (HyperFrames template)                                       │
    │  ├─ Merge todas las slides en HTML completo                                            │
    │  ├─ Save HTML → Supabase Storage                                                       │
    │  ├─ Save metadata → tracker_results                                                    │
    │  └─ Response: { ok: true, jobId, status, url (HTML storage) }                          │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~2-5m | Cost: variable

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                                 WORKFLOW 5: WEB                                         │
    │  POST /web                                                                              │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Call web-4o API (o generador custom)                                               │
    │  │  └─ Genera estructura Next.js con componentes React                                 │
    │  ├─ Generate assets (imágenes via Higgsfield)                                          │
    │  ├─ Compilar proyecto                                                                  │
    │  ├─ Deploy to Vercel (opcional)                                                        │
    │  │  └─ Retorna URL viva del deployment                                                 │
    │  ├─ Save URL → Supabase                                                                │
    │  └─ Response: { ok: true, jobId, status, url (Vercel deployment) }                     │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~5-15m | Cost: variable

    ┌─────────────────────────────────────────────────────────────────────────────────────────┐
    │                               WORKFLOW 6: CAPACITACIÓN                                  │
    │  POST /capacitacion                                                                     │
    │  ├─ Trigger: Webhook JSON                                                              │
    │  ├─ Generate educational outline                                                       │
    │  ├─ Loop (crear cada sección):                                                         │
    │  │  ├─ Contenido + diapositivas                                                        │
    │  │  ├─ Generar video intro (Higgsfield)                                                │
    │  │  └─ Generar voice narrado (ElevenLabs)                                              │
    │  ├─ Compilar HTML interactivo (quizzes, certificado)                                   │
    │  ├─ Save → Supabase Storage                                                            │
    │  └─ Response: { ok: true, jobId, status, url }                                         │
    └─────────────────────────────────────────────────────────────────────────────────────────┘
    Time: ~5-20m | Cost: variable

User: "Quiero generar una imagen de un atardecer en la playa"

    1️⃣ USUARIO abre Control Maestro
       https://tracker.victor-ia.xyz/control-maestro.html

    2️⃣ Click "🖼️ Generar Imagen"
       Modal abierto

    3️⃣ Completa formulario
       Descripción: "Atardecer en la playa con palmeras, fotografía 8K realista"
       Proporción: Horizontal (landscape)
       Estilo: Realista
       Cantidad: 1
       Click "Generar Imagen"

    4️⃣ API /api/create recibe POST
       {
         "action": "imagen",
         "voice_input": "Atardecer en la playa...",
         "config": {
           "imagen-aspect": "landscape",
           "imagen-estilo": "realista",
           "imagen-cantidad": "1"
         }
       }

    5️⃣ Genera jobId único
       "imagen-1726087234567-abc12"

    6️⃣ Guarda record inicial en Supabase
       INSERT INTO tracker_results (job_id, action, status, metadata)
       VALUES ('imagen-1726087234567-abc12', 'imagen', 'processing', {...})

    7️⃣ Envía webhook a N8N
       POST https://n8n.srv1013903.hstgr.cloud/webhook/.../imagen
       Body: { jobId, prompt, aspect_ratio, estilo, cantidad, ... }

    8️⃣ N8N Workflow "Generador de Imágenes" ejecuta
       ├─ Validate ✓
       ├─ HTTP POST → Higgsfield /v1/text2image/soul
       ├─ Recibe request_id: "1726087234567"
       ├─ Poll /v1/job-sets/1726087234567 (x15 loops, 2.5s cada una)
       │  Loop 1: status = "in_progress" → espera
       │  Loop 2: status = "in_progress" → espera
       │  ...
       │  Loop 14: status = "completed", url = "https://cdn.higgsfield.ai/img/xyz.png"
       ├─ HTTP POST → Supabase /rest/v1/tracker_results
       │  UPDATE tracker_results
       │  SET result_url = "https://cdn...",
       │      status = "completed",
       │      updated_at = NOW()
       │  WHERE job_id = 'imagen-1726087234567-abc12'
       └─ Webhook response: { ok: true, jobId, status: "completed", url: "..." }

    9️⃣ Control Maestro recibe respuesta
       ├─ Actualiza cola (status = "completed")
       ├─ Toast: "✅ Imagen lista!"
       └─ Limpia form

    🔟 Usuario abre Biblioteca
       https://tracker.victor-ia.xyz/biblioteca.html

    1️⃣1️⃣ Biblioteca fetches /api/biblioteca
       SELECT * FROM tracker_results ORDER BY created_at DESC

    1️⃣2️⃣ Resultado aparece en grid
